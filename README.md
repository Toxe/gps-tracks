# GPS Tracks

## Dependencies

- Python REST API:
  - Python 3
  - Flask
  - Flask-JWT-Extended
  - Marshmallow
  - python-dotenv
  - pytest

## Setup local development version

Python virtualenv:

```
$ cd rest-api
$ python3 -m venv venv
$ source venv/bin/activate
```

Install (or upgrade) Python packages in virtualenv:

```
$ pip install -U $(<.pip)
```
